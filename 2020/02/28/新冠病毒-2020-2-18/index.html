新冠病毒讲座--第一节

发表于 2020-02-28

抗击新冠病毒，一场全球治理的压力测试

沈逸 研究员

很难局限在一个地理范围内

很难有任何一个单一国家，完全凭借自己的力量解决这个问题

有时间压力的环境和状态的问题

跨国问题的全球治理

  • 疾病防控与全球公共卫生协同

非传统安全风险的国家处置

  • 应对信息化、全球化背景下分传统安全风险能力的国家能力

公共危机状态的决策与应急能力体系

  • 意志、战略与政策落实
  • 应急体系，危机状况下巨大的外部压力

抗击新冠病毒的背景(1)： 全球化

  • 人员跨境流动：商业等，管辖权的问题
  • 商品跨境流动：零部件的流动，工具上是否有病毒，均衡
  • 资本跨境流动：
  • 信息跨就流动：有意或无意的片面化，谣言，心理战特征的谣言
  • 全球化一体两面：更便捷的生活 vs. 更流动的风险

(2)：国际体系

  • 今天的世界仍然处于无政府状态
    • 无超国家权威
    • 西方事实上的自我中心，对合作产生很大的干扰，美国：南海自由航行行动，台湾等问题
    • 持续存在的信任赤字：国家与国家不信任，CDC数据公开透明
  • 有一定秩序的国际协调
    • 比较充分的协调机制

(3)：中国的位置 不可替代的产业链核心枢纽

  • 美国NBC指出新冠病毒测试美国医药系统对中国的依赖程度
    • 对中国有充分的信心：最核心的生产和供应
    • 紧缺是流通的问题，打通流通的环节，
    • 挤兑，平时是平稳的，突然爆发的需求量
    • 物资协调，信息协调，消除恐慌
    • 要担心中国的疫情对世界产生什么样的影响

疾病防控与全球治理

  • 1918年流感大流行，H1N1 a型流感病毒，5亿人感染，保守估计2000万人死亡，死于战争的人数高于死于战争
    • 疾病面前，人类应该团结一致

单一多家难以有效应对大规模流行性疾病的挑战，人类健康状态的持续改善离不开有效的国际协调

作为联合国专门机构的额世界卫生组织

  • “身体、精神及社会生活中的完美状态”
  • 促进流行病和地方病的防治
  • 提高和改进公共卫生，疾病医疗和有关事项的训练
  • 推进确定生物制品的国际标准

WHO的目标

  • 千年发展目标
    • 消灭极端贫穷和饥饿
    • 降低儿童死亡率
    • 改善孕妇健康
    • 与艾滋病、疟疾和其他疾病做对抗
    • 确保环境的可持续能力
    • 建立有助于促进发展的全球伙伴关系
  • 主要贡献
    • 天花
    • SARS
    • 脊髓灰质炎
    • 艾滋病

国际关注的突发公共卫生事件(PHEIC)

六次国际关注的突发公共卫生事件

  • 09：H1N1流感
  • 14：脊髓灰质炎疫情
  • 14：西非埃博拉
  • 15-16：寨卡病毒
  • 18-19：刚果埃博拉
  • 20-：新冠状病毒

这些事件都是临时性的，需要每三个月进行一次复核

  • 强调并非因为中国的疫情问题，儿是针对医疗体系较弱的国家加强疫情应对，对于中国的防疫措施仍表示肯定，并不针对中国实施旅游或贸易禁令
  • 这些显示，如果疫情发生在美国人身上，或疫情发生时不是节假日，反应会更迅速

六个问题

这次突发疫情，对国内外形势会产生什么影响

  • 这是一场压力测试
    • 对国际治理体系的压力问题
    • 对各国政府治理能力的压力测试
    • 凸显人类命运共同体建设的合理性、必要性以及迫切性
病毒面前是不分人的

宣布为PEHIC之后，如何看待各种措施

  • 有些措施是合理且可以理解的
    • 正常的国民健康保护措施是可以理解的
  • 有些措施是过度反应
    • 相应国家所在过的国内政治因素干扰
  • 有些措施或表态是借题发挥
    • 华尔街日报“亚洲病夫”

如何看待阴谋论

  • 拒绝“阴谋论”
    • “阴谋论”本质上是一种认识世界的错误方法，成因复杂，在当下用来拍悬疑剧，是可以的，但严肃的研究、分析和讨论，首先必须使用正确的方法
    • 生物科技是一门严肃、严谨、规范的科学，基因编辑以及更加尖锐的“生物武器”，不是敲打键盘，看看图片，对文字进行排列组合就可以实现的
    • 用“阴谋论”在公共卫生事件中去吸引注意力，吸引流量，或者实现其他的异途，为此不惜干扰抗击疫情的正常工作至少是一种不道德的行为

如何恰当的进行表达

  • 学会建设性的评论，与积极的建议
    • 相比03年，中国网民总数增长了10倍，移动网民总书增长了100倍
    • 表达也是一种行动，当然应该积极地进行表达，但这种表达应该是建设性的，积极的，是为了更好的抗击疫情

关于谣言的问题

  • 治理谣言、规范信息发布
  • 有效的疫情管制
  • 国家与国家之间的竞争
