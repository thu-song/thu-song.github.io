硬件和软件

  1. 硬件：组成计算机的各个物理部件，，包括CPU、主存、辅存/外存、外设
    • 主板：通过各种接口将硬件组合在一起
  2. 软件：程序，即一系列指令，计算机根据指令进行运算完成相应的任务

操作系统

  1. 操作系统：管理和控制计算机系统中的硬件和软件资源，在计算机与用户之间起到接口作用

  • 计算机硬件指的是计算机的物理部件

  • 计算机软件 = 程序 + 数据 + 文档

  • 操作系统 = 内核 + 系统程序和库

    • 内核是操作系统的核心：
      • 内核程序一直运行着
      • 驱动程序提供与硬件的接口
      • 系统调用提供用户访问内核资源
    • 系统程序和库：
      • 系统服务
      • 实用工具
      • 库
      • 提供用户界面的shell：命令行(CLI)，图形化用户界面(GUI)

  • 扩展：树莓派，微型电脑

  • wsl：Windows Subsystem for Linux，不是虚拟机，只是将Linux系统调用转换为Widows系统调用

Linux 发行版

  1. 包括：
    • Linux 内核
    • GNU 库和实用工具、桌面环境
    • 各种应用软件
    • 安装工具
    • 软件包管理工具
    • 用户手册

Linux 系统安装：虚拟机

  1. 虚拟机指的是在一台物理计算机上模拟出来的虚拟的计算机系统
    • 系统级虚拟机：完全虚拟化，在一台计算机上通过对于底层硬件的虚拟化来模拟出一台或多态虚拟的计算机，课同时运行，相互隔离
    • 进程级的虚拟机：提供一个抽象的与平台无关的程序执行环境，比如Java虚拟机
    • 操作系统级的虚拟机：通过操作系统内核的支持来支持多个分隔开来的用户空间实例
  2. 虚拟机平台所在的操作系统称为主机操作系统，而在其上运行的虚拟机称为客户操作系统
  • LTS：Long-term support

apt 命令：advanced packaging tool

  1. apt update：下载最新的package列表
  2. apt upgrate：将已经安装的package更新到最新版本
  3. apt install xxx：安装package xxx

Linux系统访问

  1. 控制台是一种特殊的终端，单用户模式以及网络无法使用时仍可使用

疑问：

  1. Linux发行版本、内核发行版本和内核版本有什么区别？
    • Linux内核是开源的，发行版的开发者下载源代码进行编译，内核版本就是开发者在编译内核时的信息，一般是描述性质的，格式可以自由指定
    • 但是内核发行版本的格式一般前面为版本号，这样可以根据其版本号进行进一步的操作
    • 比如ubuntu的最新版本使用的内核为5.3，而使用另外一个发行版比如fedora使用的内核也是5.3，那么内核发行版本会一致，但是内核版本不一样
    • 即：内核发行版 + 开发 –> 内核版本， 内核版本 + 其他工具等 –> Linux发行版本
  2. sudo shutdown -60 会发生什么？

作业

  1. 谁发明了Linux？

    a. Ken Thompson和Dennis Ritchie b. Bill Joy

    c. Steve Jobs d. Linus Torwalds

    e. Andrew Tanenbaum f. Richard Stallman

  2. 下列描述正确的是？

    a. Linux操作系统=GNU + Linux

    b. Linux操作系统的内核为开源的Darwin

    c. BSD/Unix也是Linux操作系统的一种

    d. 以上都是错误的

  3. 假设要执行sudo mycmd，请问：

    a. 要输入谁的密码？

    b. 以谁的身份执行mycmd？

    c. 接下来执行 sudo mycmd1，还要求输入密码吗？

  4. 假设用户在访问虚拟控制台3，现在要切换到虚拟控制台5，怎么样进行切换？

  5. 假设你要以tony的身份登录到IP地址为192.168.0.1的SSH服务器，该SSH服务器在端口4321上提供服务，请给出相应的ssh命令

  6. 退出shell可以使用什么快捷键？

  7. 执行sudo shutdown +60之后会发生什么？

  8. 如果想要看看Linux主机使用的是哪个发行版，应该使用什么命令？

  9. 如果你想要知道Linux主机使用的是哪个内核版本，应该使用什么命令？

我的答案

  1. d

  2. a

  3. (a) 当前用户；

    (b) 管理员的身份；

    (c) 接下来的一定时间内(默认为15分钟)内不需要重新输入密码

  4. 可以使用 Ctrl+Alt+F5(有的系统为 Alt+F5)

  5. ssh -p 4321 tony@192.168.0.1

  6. Ctrl + D

  7. 系统在60分钟之后关机

  8. lsb_release -a

  9. uname -v

注意

  1. sudo cmd ， 输入的是当前用户的密码，以root身份执行cmd
  2. 切换虚拟控制台，如果已经登录到console，则可chvt 5，但注意GUI方式是无法切换的。但是快捷键Ctrl+Alt+F5总是可以的
  3. sudo shutdown +60，注意单位是分钟而不是秒
  4. 发行版指的是ubuntu 18.04….还是centos，所以应该lsb_release -a，或者lsb_release -d 查看description。注意还可以 cat /etc/lsb-release
  5. uname -r 查看内核版本， uname -a 可以查看更多的信息。 www.kernel.org给出linux kernel的源代码，然后开发者可以获得源代码，然后针对某个硬件架构进行编译，uname -v给出了开发者编译内核时的相关信息。所以一般通过uname -r查看内核版本
