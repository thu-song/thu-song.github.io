  1. which cmd

  3. /etc/bask.bashrc和.bashrc
  4. source ~/.bashrc
  5. 将当前的时间信息附加到tmp目录下的一个文件中，文件名为tmp，后缀名为当前的日期，月-日的格式
  • 非交互式
  • 非登录shell
  • 会执行~/.bashrc
  1. mkdir -p {c,python}/{slides,homeworks,quizs}
  2. touch tmp{0..09}.txt
  3. #! /bin/bash
    read -p “请输入你的名字: “ name;
    echo “欢迎$name来到LINUX世界。”
  4. echo $(date; ls /)，打印当前的时间和/目录下的文件名，没有装引号会进行单词分割，IFS缺省情况下所有单词之间都以一个空格隔开
    echo “$(date; ls /)”，打印当前的时间和/目录下的文件名，会保留原有的(date; ls /)输出的格式打印在屏幕上，不会进行单词分割处理
  • 如果foo1执行失败，那么之后的命令都不会被执行；最终状态码为1
  • 如果foo1执行成功，foo2会被执行，并且foo2的输出内容将通过管道输入到foo3，之后将foo3的输出会被重定向到foo4文件中；最终状态码取决于foo3，如果foo3执行成功，状态码为1，否则为0
