新冠病毒讲座--第二节

发表于 2020-03-03

新冠疫情下的情绪管理 —— 高隽

新冠疫情是什么：危机与情绪反应

对于情绪，首先请觉察你对情绪的态度

- 你对情绪态度直接影响你的情绪体验

好奇是通向智慧的道路

- 情绪是一组复杂的身体和心理反应模式
- 每个“情绪礼包”通常会包括：(1)生理唤醒 (2)主观的感觉 (3)认知过程 (4)外显的情绪表达 (5)特定的行为反应
- WHY：往往是针对**有个人意义**的情境或事件的**应对反应**

情绪是有功能的

- 情绪有着重要的个人、人际和社会功能
    - 在个人层面：在当下处境完成迅速评估，激发具体的应对反应，影响决策
    - 在人际和社会层面：沟通信息，调节人际关系和群体行为
        - 共情与利他行为
        - 情绪传染与群体事件

每个人都需要学习情绪管理这一课

- 情绪本身并无好坏优劣之分
- 如何体验与调节情绪则有“适应”和“不适应”的差异
- 个人层面的情绪管理
    - 不要因为自己有情绪而有情绪

情绪管理的散步：觉察、理解、行为

- 情绪管理的三个基本步骤
    - 觉察和命名你的情绪体验
    - 理解你的情绪体验总是试图在传递给你一些信息，然后思考一下到底是什么信息
    - 有意识地做出具体的行动：实现特定情绪的“功能目标”，调节情绪的强度
- 三个现象与三条建议
    - 现象1：强烈情绪体验(尤其是胜利唤起)下，高级认知功能会出现“死机”
    - 建议1：要先安身，再安心，慎做重大的决策
    - 现象2：情绪体验的不同成份相互影响
    - 建议2：相对最容易改变的成份是行为
    - 现象3：积极情绪被发现有诸多益处
    - 建议3：请主动、有意识地创造和使用积极情绪

我们还能从“新冠情绪危机”中收获什么？

- 危机事件可能会暂时或持久地影响
    - 安全感、控制感、信任感
    - 人生观、世界观、价值观
