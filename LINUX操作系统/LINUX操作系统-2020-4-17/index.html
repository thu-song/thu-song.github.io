LINUX操作系统--第八节(编辑器 vim)

发表于 2020-04-10

题目

下面的命令指的都是vi命令。

  1. 在正常模式，你应该怎么快速地移动到当前行的行首和行尾？

  2. 你要在当前行上方插入一行新的内容，应该使用哪个命令?

  3. 你想要在当前行的行尾附加text，请给出尽可能简洁的步骤。

  4. 你想要将当前行开始连续3行合并为同一行，请给出相应的命令。

  5. 在正常模式下应该使用什么命令快速地移动到当前行的第5个word?，然后将其前面4个字符替换为counter。

  6. 你发现在当前文件中不小心将Computer误写成了Comptuer，光标当前位置正好在Comptuer这个单词中的某个字符上面，现在想要将文件中的Comptuer单词都修改为Computer，请给出相应的步骤。暂时不要使用我们尚未介绍的:substitute命令，假设set ignorecase smartcase已经设置，在这个过程中还应该设置哪个选项，以使得搜索到文件两端时自动回绕？

  7. 下面是一行文本,当前光标在or的第一个字符上，
    text = “Leading or trailing white space is included, but not counted.”
    你想要将这一行替换为如下内容，请给出相应的步骤，越简洁越好
    text = ‘foo’

  8. 你要把从当前行开始的4行删除，然后将其插入到文件最前面，请给出相应的步骤。

我的答案

  1. 行首：0或
    行尾：$或
  2. 正常模式下 O 进入插入模式，之后在光标处插入即可
  3. 正常模式下 A 进入插入模式，之后在光标处插入 text 即可
  4. 3J
  5. 移动光标：首先 0 将光标移动到行首，之后 4w 移动到第 5 个word
    替换：c4bcounter
  6. ① b 移到单词的开始位置
    ② cwComputer 修改为 Computer
    ③ /Comptuer 查找 Comptuer
    ④ . 重复上次的修改
    ⑤ 重复③④直至找不到 Comptuer

设置选项：set wrapscan
7.
① ca” ‘foo’
8.
① 4dd
② gg
③ “”P

参考答案

  1. 在正常模式，你应该怎么快速地移动到当前行的行首和行尾？
    0移动到行首
    $移动到行尾

  2. 你要在当前行上方插入一行新的内容，应该使用哪个命令
    O 上一行插入， o为下一行插入

  3. 你想要在当前行的行尾附加text，请给出尽可能简洁的步骤。
    Atext 或$atext

  4. 你想要将当前行开始连续3行合并为同一行，请给出相应的命令
    3J
    不是2J

  5. 在正常模式下应该使用什么命令快速地移动到当前行的第5个word?，然后将其前面4个字符替换为counter。
    ^移动到行首的第一个非空格字符，4w往右移动4个word，这样是第5个word，c4l替换前面4个字符，输入counter

注意不能用4rc，表示重复4次rc，替换为4个c

  1. 你发现在当前文件中不小心将Computer误写成了Comptuer，光标当前位置正好在Comptuer这个单词中的某个字符上面，现在想要将文件中的Comptuer单词都修改为Computer，请给出相应的步骤。暂时不要使用我们尚未介绍的:substitute命令，假设set ignorecase smartcase已经设置，在这个过程中还应该设置哪个选项，以使得搜索到文件两端时自动回绕？

因为smartcase已经设置，这样搜索Computer会大小写相关，已经在单词里面，这样*表示搜索当前光标所在单词,然后cwComputer，然后n重复搜索，dot(.)重复上次替换，一直这样，直到没有匹配项为止。
set wrapscan

  1. 下面是一行文本,当前光标在or的第一个字符上，
    text = “Leading or trailing white space is included, but not counted.”
    你想要将这一行替换为如下内容，请给出相应的步骤，越简洁越好
    text = ‘foo’

使用文本对象的around形式
ca” ‘foo’
文本对象aw/iw等前面必须有操作符，或者在可视模式，否则解释为append了

同学们的如下版本也可以：
2b或B或F”跳到”，然后cf”替换为’foo’
2b跳到”，然后C替换到行尾’foo’

要替换，不用首先d然后i

  1. 你要把从当前行开始的4行删除，然后将其插入到文件最前面，请给出相应的步骤。
    4dd或d4j，gg到第一行，然后P插入到第一行之前

在以行单位操作时，光标在这一行的任何位置都可以

注意和其他编辑器不一样，删除的内容也会保存到寄存器中。

另外应该是d4j而不是d3j，我看了一下应该是上次课件里面的例子有误，dj仅仅删除当前行。
操作符命令 + motion，motion有line-wise和character-wise的概念，即以行为单位，还是以字符为单位，还有inclusive和exclusive，即是否包含光标所在的字符或者行。比如hjkl等都是exclusive的，这样dj仅仅删除当前行，w是exclusive的，这样不会删除下一个单词的首个字符，而e是inclusive的，这样de会删除到单词结尾，包括结尾处的单词。大部分情况下不用特别关注是否是inclusive和exclusive，遵循的应该是大家很自然的习惯。
