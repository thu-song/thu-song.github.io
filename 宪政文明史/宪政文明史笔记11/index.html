宪政文明史笔记--第十一周

发表于 2019-11-21

(二)美国宪法

  1. 1791年，一次性通过了十条宪法修正案，权利法案
    • 个人自由权利：生命财产、集会、请愿、
    • 不得强迫任何人自证其罪，保持沉默权（米兰达权利/规则）
    • 联邦和州的分权
    • 12条：总统选举方式
    • 13、14、15条：南北战争期间
    • 16条：选举
    • 18条：禁酒令，一战，粮食造酒
    • 19条：妇女有选举权
    • 20条：总统就职，3月4号–>1月20号
    • 21条：废除禁酒令（18条），带动就业
    • 22条：总统任期，2届，禁止总统第三次选举
    • 27条：男女平权案（现是废案）
    • 28条：哥伦比亚特区改州
    • 不变应万变，通过法院解释，总统的言论，使得法律的精神随着时代变化而变化
  2. 彻底的共和–相比英国，使得共和政体成为可以操作的现实，美国宪法对人类社会的贡献
  3. 美国宪法的特点：
    • 文本简练
    • 人民主权：人民是首要的
    • 分权制衡：权力之间的制衡，横向的：司法、行政、立法，纵向的：联邦和州
    • 有限政府：分权制衡的理论基础就是有限政府，政府的权利是有限的，人民保留了天然的权利

(三)法国宪法

  1. 大陆法系国家中制定宪法最早，颁布宪法最多的国家，平均大概十年左右，就有一部新宪法
  2. 《人权宣言》，第一个宪法文件，拉开了法国近代运动的序幕
    • 全称《人权和公民权宣言》，借鉴了《独立宣言》的内容
    • 核心：人民基本权利的保障
    • 人权范围广：言论自由等
    • 法律原则：法律面前人人平等，法无明文规定不为罪，罪刑法定，无罪推定，私有财产神圣不可侵犯都来源于《人权宣言》
  3. 1791年宪法，正文部分确立了人民主权原则，三权分立的君主立宪制等

第四章 现代宪政文明的发展

第一节 现代宪政思想

  1. 时代背景：
    • 消极国家观，国家只要维持基本的秩序，把个人自由和国家权利对立起来的古典自由观
    • 20世纪，贫富不均造成的社会问题日趋严重，社会矛盾日益严重，强调国家要积极执行权力义务，保证每个人都有基本生活权力

一、社会民主主义思潮

  1. 资产阶级革命胜利，小资产阶级仍在压迫之下
  2. 社会主义主要立场：把平等公正放在和自由同等重要的地位，每个社会成员尽可能平等的社会地位和公正；经济民主、社会民主
  3. 积极自由观：内涵：如果某些机会和资源是实现自由所必需的，那么拥有这些机会和资源本身就是自由的一部分，强调政府要创造公民享受自由的物质和社会条件，国家对社会经济环境的干预

二、个人主义复兴

  1. 对国家权力的反对，对个人自由权力的强调

(一)哈耶克宪政思想

  1. 强调自由市场次序，诺贝尔经济奖
  2. 复兴古典自由主义，哈耶克关注的重点：怎么理解个人和社会之间的关系，在政治领域，如何保护个人自由主义和维护社会次序间的平衡
  3. 偏向个人主义
    • 个人主义的基本特征是尊重个人为人
  4. *论述了自由和法治之间的关系
    • 自由是完全不受他人专横所控制的私人领域，需要法律保护
    • 法制是一种理想，效率是前瞻的，确定的，可知的东西，平等的
    • 模范宪政的方案：良宪和分权

(二)罗尔斯宪政思想

  1. 《正义论》，二战之后西方最有影响力的哲学思想家之一
  2. 正义理论：
    • 正义对社会制度来说是至关重要的，正义是法律、社会主义制度的首要价值
    • 正义是人民选择的结果，
  3. 正义有第一原则和第二原则：
    • 第一原则：平等的自由原则，对所有人都是一样的，每个人都享有与其他人一样的自由
    • 第二原则：机会均等原则和差别原则的结合，允许一些不平等的安排，使得不平等的安排对每一个人都是有利的，每个人的出身、天赋不同，对天赋较低的人有一些关注，纠正偶尔因素造成的偏差，实现平等
  4. 怎么实现正义原则：
    • 选择正义原则
    • 立宪阶段，好的制度
    • 立法阶段
    • 适用法律
  5. 对法治的平等和自由的论述
    • 法制自由的保障：个人的自由必须受到法制的保障，认真贯彻法制的原则对自由的威胁才会减少
    • 只有人们在法律规定的范围内活动，自由才不会受到侵犯

第二节 现代宪政制度

  1. 社会民主主义宪法产生于20世纪初
  2. 1918年，苏俄宪法是人类历史上第一部社会主义类型宪法

一、魏玛宪法

  1. 德国
  2. 1918年爆发革命，《德意志共和国宪法》，史称魏玛宪法
  3. 标志：近代宪法向现代宪法转变的标志性的文件，现代宪法产生的标志
  4. 联邦的组织和职责，德国人民的基本权利和义务
    • 联邦制国家
    • 规定德国实行共和政体，分权原则组织政府
    • 宪法对公民的权力义务做了全面的规定，增加了公民在经济社会劳动等方面新的权力，社会经济文化权力
    • 特别的，对社会经济生活做了专门的规定，首次将公民权由社会领域扩张到经济领域，实现了人权由自由权到经济权的转化，对私有制也做了一些限制
    • 因此成为经济宪法
  5. 体现了国家积极干预公民生活，为公民实现自由提供必要的条件，所以开创了社会主义宪法的先河，从消极自由到积极自由的转变，有利于对人权的全面保障

二、罗斯福新政

  1. 宪法意义上，通过对自由主义的重新阐释，体现社会民主主义立宪思想
  2. 以凯恩斯理论为基础
  3. 经济危机必然引起政治危机
  4. 向国会提供了一系列法案，内容设计工业、劳动等，强调政府对经济社会的干预，扩大联邦政府的权力，强调在社会经济生活的干预
  5. 对自由主义重新阐释，积极自由观，自由除了自由本身之外，还有经济等外在的资源和保障，使得美国立宪主义理念内涵得到扩展，即社会经济文化权力当作自由权来进行改造
  6. 现代宪法的特征，承认国家干预的合理性
