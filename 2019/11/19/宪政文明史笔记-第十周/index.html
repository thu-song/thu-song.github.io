宪政文明史笔记--第十周

发表于 2019-11-19

#近代宪政文明

（四）卢梭宪政思想

  1. 对洛克的社会契约论进行改造
  2. 公意：公共的意志，永远都是公正的，所有人都同意的意志；众意：简单的个人意志相加；
  3. 社会契约论：
  4. 人民主权理论：把所有的人视为人民，都有参与政治的权利，主权就是共同体的最高权力
    • 人民的公意，在国家中表现为最高权力，主权是公意的具体表现
    • 人民是国家最高权力的来源，主权属于人民
    • 主权的基本特性：不可转让（转让主权相当于转让意志），不可分割（人民整体的公共意志）

（五）杰弗逊宪政思想

  1. 亚当斯，美国第三任总统，起草独立宣言
  2. 思想：
    • 人人都有天赋权利（自由，平等）
    • 政府存在的正当性就是保障这些权利
    • 程序的正当性：政府的正当性要经过被统治者的同意
  3. 构建共和政府的两大原则：主权在民（普选，监督权）；制权的分立和制衡（联邦：立法、行政、司法（横向的三权分立）；联邦和地方（纵向的中央和地方））
  4. 人民行使权利时：少数服从多数的共和原则；多数尊重少数的宪政原则
  5. 持久的宪政次序要有民众基础，要培养适当（适合民主社会）的公民素质–国家教育的根本任务

（六）哈密尔顿宪政思想

  1. 华盛顿政府的财政部长
  2. 主张人性本恶，国家存在之前，人与人处于战争状态，需要政府制约，“如果人人都是天使，那就不需要任何政府了；如果天使统治人，就不需要对政府有任何内在的或外在的制约”
  3. 精英

###近代宪政思想的成熟

（一）约翰密尔

  1. 近代宪政思想到现代宪政思想的成熟
  2. 修正功利主义理论，个人的快乐幸福以他人的快乐幸福为前提，
  3. 政府有责任教育公民追求更高尚的幸福生活
  4. 自由主义理论：
    • 权任，政府不能采取放任主义确保公民的自由，要有积极的行动，立法成为创造机会和使机会均等的手段，为了最大多数人的幸
    • 通过限制政府权力保障人民权利，发挥政府在促进
    • 代议制，产生最好的法律

（二）弓斯当

  1. 坚持人民主权的情况下，强调不可侵犯的人民权利
  2. 古代人的自由：人人都有自由参与公共事务的讨论与决策
    现代人的自由：享有不受政府保护的个人自由，不能亲自参与公共事务的讨论

（三）托克维尔

  1. 法国
  2. 考察美国民主–《论美国的民主》上下
  3. 围绕民主：
    • 肯定个人主义思想，承认民主制的优越性和不可否认的发展趋势
    • 民主制的缺陷–多数人的暴政
  4. 克服民主的缺陷，从精神：理解自利的原则–有德、克制、温和；从制度：政府有责任培养公共精神

第二节 近代宪政制度

一、近代宪法产生的历史条件、历史必然性

  1. 经济：在经济上具备了指定基本法的条件；资本主义生产关系的确立，商品经济关系奠定了资本主义宪法产生的前提；
    • 决定了人与人的公开的不平等；宪法是对民主事实的确立
    • 等价交换有利于平等观念的普及
    • bala bala
  2. 政治：对社会实行独立的统治，资产阶级掌握国家政权，资产阶级民主制度的确立是宪法产生的政治条件
    • 封建专制：不可能存在以平等为基础存在的宪法
    • 代议民主制，没有最高权威，所有人都是平等的，按照资产阶级的整体意志，指定总章程，按照总章程管理国家
  3. 思想文化：三权分立，天赋主权、人民权利等思想深入人心，是宪法产生的思想条件
    • 自然权利，天赋权利，国家的目的是保证人民的天赋主权
    • 从人民主权，人民政府不能专制
    • “专制国家中，国王就是法律；同样的，自由国家中，法律便成为国王”
    • 从人民主权确立了法制
  4. 法律本身： 法律制度本身的发展，法律形式的分化是宪法产生的法律条件
    • 古代：诸法合体，以刑为主
    • 商品经济，使得民事法律发展起来

二、英美法宪法

（一）英国宪法

  1. 商品经济发达，资本主义商品关系形成，但是仍有贵族存在
  2. 英国是近代宪法的发源地，英国宪法是近代宪法之母–英美法系的基础
  3. 没有成文宪法典
  4. 1689年，权利法案，标志着君主立宪制在英国的确立：限制王权，强调议会至上
  5. 1701年，王位继承法，用法律规定王位继承的机制
  6. 不仅产生最早，而且有特色：渐进性、延续性、适应性；都是经过长期积累逐渐定型；
    不是彻底否定旧的宪法渊源的基础上建立起来的，而是继承旧的宪法渊源，重新阐释，新的宪法原则和精神在旧的法律渊源的基础上产生，并且发展

（二）美国宪法

  1. 世界上第一个成文宪法
  2. 州宪是美国最早的一批成文宪法，是根据人民主权指定的宪法，对人民主权的保障
  3. 《独立宣言》
  4. 反对君主立宪制，独立战争前后，11个州指定宪法
  5. 分权原则、两院制度、法官终身都起源于州宪
  6. 美国宪法：1序言+7正文
    • 序言：美国宪法来自人民
    • 州宪法和联邦宪法的关系
    • 规定宪法经过9个州的批准生效
  7. 早期宪法中没有公民权利的保证
    • 各州已经有权利法案，不需要再对人民权利进行规定
    • 宪法本身是对政府权力的规定，之外就是公民权利自由的空间
  8. 1791年，前十条修正案
