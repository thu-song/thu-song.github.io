宪政文明史笔记--第九周

发表于 2019-11-19

第二节 中世纪宪政文明

  1. 中世纪影响宪政的原因：反宪政因素少
    • 中世纪的欧洲，没有压倒一切的绝对的权利，产生了个人的自由和权力；多元和妥协
    • 中世纪的教会势力，教会的权力和世俗的权力，各自对立，彼此制约–>国家权力有限；
      • 借助对宗教的信仰，来提升法律的权威
      • 基督教的原罪论，人性本恶，政府的权力需要加以限制；
      • 平等观念的传播
    • 没有强烈的爱国主义，关注个人权利、自由，对国家权力的防范
    • 法治，中世纪不缺少法制（上帝的法）– “法律是需要被信仰的，否则将如同虚设”

一、托马斯·阿奎那宪政思想

  1. 神学主义思想家
  2. 自然法理论：
    • 渊源：自然主义特征，法律是自然的延申；法的二元论，发有自然法和人订法之分；自然法是正义的基础；不符合自然法的人定法都是不合理的
    • 内容：宇宙是永恒法、自然法（强调良法，对人定法的评定确立一种标准）、神法（一些不重要的自然法不会被注意到，因此需要神法的启示；人不仅要追求今生的幸福，还要追求来世的永生）、人订法（管理，合理的人定法是符合自然法的，是对现实的理论的道德评价）的四种法的体系，最高法是永恒法，在人身上的体现是自然法
      • 神学色彩；与西塞罗的自然法思想
  3. 君主制：
    • 君主制是最适合社会政治生活的制度，最接近自然的形式，支配权总是掌握在某个人的手里
    • 如何防止君主制沦为暴君政体？需要道德、法律、制度的约束；成为君主前要道德考察，成为君主之后要受到自然法和人定法的双重约束（人民有权利反抗）

二、博丹宪政思想

  1. 人物评价：思想史上、宪政史上第一个系统阐述国家主权理论的人
  2. 家庭两个特征：财产、涵盖了合法的君主的权威和完美的政府之间的原型；
    • 妻子、子女不能决定身份–>公民
    • 父亲具有绝对的支配权，体力和理性–>国家应该具有绝对的权威（主权）
  3. 主权理论
    • 主权的特性：最高性（对内：派生国家内部的一切权力、对外：独立的、自由的）、不受限制的（不能侵犯个人私有财产）、永久性（主权者永久地保留主权，可以委托与收回）、不可分割和不可转让、……

第三章 近代宪政文明的产生

前言

  1. 思想部分和制度部分
  2. 强调：政体的混合与权力的均衡
  3. 社会契约论、分权理论
  4. 社会契约说（为政治的合法性奠定了基础）和分权说（针对专制政体、是对专制集权的彻底否定）
  5. 近代的宪政理论是最纯粹、最经典的宪政理论
  6. 通过制度设计解决现实的问题
  7. 近代宪政文明的产生在公元1500年前后

第一节 近代宪政思想

  1. 最纯粹的、最经典的宪政思想
  2. 背景：
    • 文艺复兴、宗教改革、启蒙运动
    • 个人主义思想，经过重大的事件，逐渐成为社会上主导的社会观念，人类社会进入现代化
      • 文艺复兴：把人作为一个类别、整体；人的发现、复兴，人文主义运动，强调人的价值、地位、尊严，人类的张扬，人性的觉醒
      • 宗教改革：开始关注独立的、自主的个人，强调宗教的个人主义；突出个人意识，eg：教徒、个人直接与上帝沟通，得到上帝的救赎，把个人的良心置于一切外在的权威之上；人不能改变命运，必须孤独地走在人生旅途中，没有依靠，独立，个人良知成为判断的标准
      • 启蒙运动：宗教个人主义–>哲学个人主义、政治个人主义（国家作为一个整体，是由人作为一个个体的基本要素组成，从个体出发建构政治社会–>社会契约论（霍布斯理论））；“我思故我在”，使人的认识基础因人而异，奠定了彻底的个人主义的基础；个人启蒙思想的诞生

（一）霍布斯宪政理论

  1. 英国，似是而非，近代立宪主义的起点
  2. 著作：《利维坦》
  3. 提出社会契约的基础：
    • 自然状态是险恶的：自然状态下，人与人之间是狼的关系，始终处在战争状态，没有法律、道德，获取利益
    • 比起自然状态下的战争，人是有感情的，更趋于和平，理性为和平提供了自然法
    • 只要有战争存在，自然法无法实施；缔结契约以实现自然法的实施，依靠契约，所有个体交出所有权力，形成主权权力
  4. 社会契约论
    • 转折意义，
      • 传统的政治哲学观念：整体先与个人，个人无条件服从整体；
      • 人先于整体而存在，个体先于政治社会的出现，个人权利是法律的基础、政治社会的基础

（二）洛克宪政思想

  1. 英国自由主义思想家，奠定了近代立宪主义的基础
  2. 思想基础：
    • 自然状态，政府保护个人利益
    • 自然状态：
      • 常态：自由、平等的生活状态，但是充满恐惧，因为自然状态的缺陷–缺少法律，缺少公正的裁判者，缺少使得判决得以执行的权力；
      • 非常态：战争状态，威胁其他人的权力
    • 需要通过权力制约，订立契约
  3. 与霍布斯的不同：每个人保存最基本的权利–生命、自由、财产；立法、执法等权力由国家、政府承担
  4. 通过自然状态、法、权力，先验性概念演绎他的理论，表达基本的宪政思想：
    • 自然权利是基于人、人之所以为人所固有的天赋权利，人人享有自然权利
    • 国家政府的存在就是维持一个公正的社会次序（工具性的价值，责任政府的理论基础），确保个人安全地享受个人权利
    • 政治状态基于克服自然状态的缺陷而存在 –> 公正的法律、公正的司法官、使得判决得以执行的执法力量；法治政府、责任政府

（三）孟德斯鸠宪政思想

  1. 法国，三权分立宪政体制的理论奠基人，最早系统的阐述了三权分立的理论
  2. 著作：《论法的精神》
  3. 与洛克相比：强调政治自由，保障自由的宪政环境，补充了洛克的宪政思想
  4. 思想：
    • 强调政治自由：与法治联系在一起，自由是做法律所许可的一切事情的权力，一个公民如果可以做法律所禁止的事情，那么就不再自由，，，
      一个人能够做他应该做的事情，而不能强迫他不应该做的事情
      一个人有权利的人都容易滥用权力，防止滥用权力，必须以权力约束权力
      最靠的政府形式：立法、行政、司法三权分立相互制约，达到平衡

孟德斯鸠对现代宪政思想最大的贡献？
- 强调保障自由的宪政环境

（四）卢梭宪政思想

  1. 法国
  2. 著作：《论人类不平等的起源和基础》《社会契约论》
    政府权力的正当性和合法性问题
