宪政文明史笔记--第十二周

发表于 2019-11-28

第五章 世界文明宪政文明与中国宪政建设

第一节 中国宪政文明史

背景

  1. 近代中国的宪政思想是从西方传过来的
  2. 在探索西方富强中学习，救亡图存
  3. 特定的国情，国家富强、民族复兴 –> 人民幸福
  4. 近代文明的宪法就是富强宪法
  5. 特定的国情决定了宪法从一开始就是为了摆脱民族的危机，把国家的强大放到了第一位
  6. 西方文明对中国文明的渗透和传播的历史
  7. 戊戌变法之后，“宪政”

一、梁启超宪政思想

  1. 宪法的概念的区别：
    • 广义：所有国家的宪法
    • 狭义：有议会国家的宪法，人民主权国家的宪法
  2. 强调宪法至上，根本行为准则
  3. 宪政就是立宪政治，国家权力是否受到限制，控制政府权力是立宪政治的精神特质
  4. 主张君主立宪制，是政体之最良者
    • 民主共和不可：国民缺少政治常识、意识、能力、自制能力

二、孙中山宪政思想

  1. 五权宪法论
    • 开始有中国的意识和实现
    • “破天荒”，创新
    • 五权分立：司法、行政、司法、考试（选拔官吏）、监察（监督官吏）
    • 人民有权：国家主权属于人民（四大政治权利：选举、罢免、创制，广泛的权利和自由）
    • 政府有能：管理政府的人是专门的人才
    • 权能分制：主权是人民的，制权是政府的

三、清末、北洋政府和南京国民政府的宪政

  1. 清末立宪：
    • 1905年起
    • 《钦定宪法大纲》
      • 中国历史上第一个宪法性文件，基本抄袭日本宪法
      • 对皇权的限制
      • 评价：从无到有，走出了第一步；晚清颓势骗局；多少对皇权做了限制，维护国家统一
  2. 北洋军阀政府时期：
    • 辛亥革命失败后
    • 特点：军阀混战，政权更迭频繁，对应立宪频繁
    • 《yuan ji yue fa》
    • 《cao kun xian fa》 | 《hui xuan xian fa》
      • 第一部政治宪法
  3. 南京国民政府宪法：
    • 正式的中华民国宪法，

四、资产阶级民主共和国宪政

  1. 《中华民国临时约法》
    • 中国历史上唯一一个资产阶级性质的宪法性文件
    • 评价：辛亥革命的产物，美国宪法为模式，开创了中国宪政的新局面，在20世纪的亚洲影响很大
    • 强调主权在民，确立人人平等、三权分立等资本主义民主思想

五、新民主主义宪政

  1. 革命根据地
  2. 《中华苏维埃共和国宪法大纲》
    • 1931，在瑞金宣告中华苏维埃共和国诞生
    • 作为根本法
    • 国家性质、各民族平等等
    • 第一个人民民主性质的宪法文件

六、社会主义宪政

  1. 1949年2月，中共中央，《废除国民党的六法全书和确定解放区司法原则》
    • 六法：宪法 刑法 民法 刑事诉讼法 民事诉讼法，，
  2. 6月，毛《论人民民主纲要》
    • –>《共同纲领》

（一）共同纲领，中央人民政府组织法

  1. 临时宪法
  2. 今年：政协70年
  3. 前：相当于宪法的序言、总纲部分、权利义务部分
  4. 后：相当于宪法的国家机构部分

（二）五四宪法

  1. 结束了新民主主义革命的历史任务，完成了社会主义改造
  2. 现行宪法日：12.4
  3. 序言、四章、106条
  4. 评价：中国宪法史上第一部社会主义类型的宪法，建立社会主义制度，对人民宪法意识的提高发挥了很大的作用
  5. 没有发挥很大作用的原因：
    • 本身定性：过渡时期，社会主义改造
    • 领导人（毛）的宪法意识不够
    • 没有保证宪法实施、监督宪法实施的机制
    • 完成共产党的主张

（三）七五宪法

  1. 之前：反右、文化大革命
  2. 评价：
    • 指导思想：左，阶级斗争为纲
    • 内容：错误的，写在宪法中不合适，中国共产党、军委主席统领中国武装力量等（正：中国共产党领导是，，最本质的特征）
    • 表现形式：宜短不宜长，深挖洞、广积粮、不称霸等
  3. 也是社会主义类型的宪法

（四）七八宪法

  1. 78年3月，社会主义宪法
    • 恢复了检察院的建制等
    • 增加了新的内容
    • 指导思想还是左的，两个凡是
    • 阶级斗争为纲
  2. 两次宪法修改，以决议方式修改

（五）八二宪法

  1. 十一届三中全会，工作重心转移；七八宪法不适应社会发展
  2. 中共中央，修宪委员会
  3. 全民讨论，提议
  4. 结构发生变化
    • 序言，总纲，公民基本权利义务，国家权力机构
  5. 修改
    • 88年
      • 两条修正案：土地使用权转让，私营经济合法
    • 93年
      • 9条
      • 92年南巡讲话
      • 6条：市场经济
      • 国营–>国有化，为股份制等提供依据
    • 99年
      • 6条
      • 主要内容：依法治国，建设社会主义法制国家
      • 邓小平理论指导地位
      • 坚持公有制为主体，，，
      • 97年，刑法更改危害国家安全罪（宪法是母法，其他法律是子法）
    • 04年
      • 14条
      • 主要内容：国家尊重和保障人权、公民的合法私有财产不受侵犯
      • 之前有国旗国徽，现增加国歌
      • 此时31条
    • 18年
      • 21条，共增加52条，现行宪法143条
      • 增加监察委员会
      • 原则：
        • 坚持党的领导，党对宪法修改的领导
        • 64条，严格依法按程序推进宪法修改
        • 充分发扬民主共识
        • 坚持对宪法部分修改，不做大改
      • 写入习近平的思想（毛泽东思想、邓小平理论）
      • （富强民主文明）+和谐美丽
      • 宪法宣誓

修宪中的一些问题

  1. 中共中央的一些建议，全国人大常委会讨论转化为宪法修正案草案，提交全国人大，宪法修正案
    • 54年，没有修宪程序，毛领导
    • 75，中共中央，，，
    • 82，中共中央建议，成立宪法委员会
  2. 全民讨论：人民当家做主
  3. 个人影响：毛，邓，习
  4. 修正案的表决：弃权票、废票、丢失票
